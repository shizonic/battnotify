# battnotify

```sh
$ ./battnotify --help
battnotify 0.1.0
shizonic <realtiaz@gmail.com>
Leightweight battery low notifier

USAGE:
    battnotify [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -i, --interval <INTERVAL>    Sets the interval in seconds to check the remaining battery
    -l, --limit <LIMIT>          Sets the limit in minutes remaining on which notifier is triggered
    -m, --message <INTERVAL>     Sets the body of the notification message
    -t, --title <TITLE>          Sets the title of the notification message
```
